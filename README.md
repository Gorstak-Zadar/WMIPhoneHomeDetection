# 📞 WMIPhoneHomeDetection

> **WMI C2 Phone-Home Detection** - Detects WMI-based command and control communication.

---

## 📋 Overview

WMIPhoneHomeDetection monitors for WMI-based command and control (C2) communication where attackers use WMI for stealthy communication.

---

## 🎯 What It Detects

- 📞 **WMI C2** - WMI-based communication
- 🌐 **Phone-Home** - External callbacks
- 🔄 **WMI Events** - Suspicious event usage
- 🕵️ **Covert Channels** - Hidden communication

---

## 🚀 Usage

```powershell
# Detect WMI C2
.\WMIPhoneHomeDetection.ps1
```
